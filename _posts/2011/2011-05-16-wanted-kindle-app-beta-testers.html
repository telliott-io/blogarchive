Own a Kindle? Interested in trying out a new productivity app? Get in touch! I’m currently looking for a small number of beta testers for a note taking and mind mapping tool called Noteree.

UPDATE: Thanks for everyone who's volunteered to take part in the Noteree beta. I've now filled all the spaces I had available. If you're interested in keeping up to date on now Noteree's coming along and when it will be released, take a look at the dedicated Noteree site.

I've managed to snag myself a place on the Kindle SDK beta programme and I've been hammering away at my first Kindle application over the last few weeks. I've got a nice stable build together, so now I need you to help me test it. I wanted to write an app that I'd want to use myself, that would also fit the nature of the platform, and the result of this is a simple little note taking application for writers: Noteree,

What’s Noteree?

Noteree is a Kindle application for generating and keeping ideas. Write simple text notes anywhere and anytime inspiration strikes.

Need a little kick getting that inspiration? Noteree includes a mind mapping tool to help build ideas from the ground up.

Features include:

  • Free tagging of notes
  • Search notes by text or tag
  • Export notes as text
  • Export Mind Maps to SVG (readable in any web browser or vector graphics application)

Plus there are plans for lots more exciting features in future versions!

What do I need to do?

I’m after some beta testers to try out the current build of Noteree to help weed out any niggling bugs that might be lurking in the code.

Ideal testers would be anyone who has to keep a lot of notes for their work or hobby (writers, comedians, public speakers, etc). You’ll also need to have your own Kindle, any version will do.

If you come across any bugs or confusing behaviours while using Noteree, I’d appreciate a note describing your problem as early as possible. I’ll be using Bugzilla to track issues, so it’ll be nice and quick for you to raise issues.

At the end of the beta period (aiming to finish 5th June), you’d also need to send me a short report with your impressions of the app.

What’s in it for me?

Anyone helping out with testing will get a copy of Noteree (naturally), and a copy of the final version - the top-tier if the release is split.

As an extra thank you, I’ll also send each tester a £10 Amazon voucher after their feedback has been received.

How do I sign up?

If you’re interested in taking part, just drop me an e-mail at tom.w.elliott@gmail.com with a little about yourself and how you might use an app like Noteree. I'll only be able to issue keys to a few devices, so don't delay!
